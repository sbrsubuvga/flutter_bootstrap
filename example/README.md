This is an example Flutter project demonstrating the usage of the `bootstrap` package.

## How to Run

1. Navigate to this directory:
   ```bash
   cd /Users/lithos/APPS/SBR/FLUTTER/PACKAGES/bootstrap/example
   ```

2. Get the dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
